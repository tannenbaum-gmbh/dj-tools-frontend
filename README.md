## Repository Overview

This repository represents the main customer-facing website for our DJ tools shop, featuring:

- **Product Catalog**: Comprehensive listings of DJ equipment, software, and accessories
- **Smart Recommendations**: AI-powered suggestions based on user preferences and DJ style
- **Interactive Features**: Virtual setup builder, compatibility checker, product comparisons
- **User Experience**: Responsive design, accessibility features, multi-language support

## Technology Stack

- **Frontend**: React 18, Next.js 14, TypeScript
- **Styling**: Tailwind CSS, Framer Motion
- **State Management**: Zustand, React Query
- **Testing**: Jest, Cypress, Testing Library
- **Deployment**: Vercel, GitHub Actions

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn package manager
- Git

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/tannenbaum-gmbh/dj-tools-frontend.git
   cd dj-tools-frontend
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Set up environment variables
   ```bash
   cp .env.example .env.local
   # Edit .env.local with your configuration
   ```

4. Run development server
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run test` - Run tests
- `npm run lint` - Run linter
